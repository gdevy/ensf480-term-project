To compile and run the client:
	- Navigate to 'src'
	- Compile using ./compProj.sh
	- Run using 'java client.userInterface.GUIController'

To compile and run the server (**Note the computer with the server must have the sqlite database configured):
	- Navigate to 'src'
	- Compile using ./compProj.sh
	- Run using 'java server.Controller'
